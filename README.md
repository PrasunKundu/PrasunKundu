## Featured Projects

### [Page Roast](https://page-roast-psi.vercel.app) — AI Landing Page Analyzer
> *Full-stack AI SaaS. Built and shipped solo.*

Captures live screenshots via Browserless.io → routes through Gemini 1.5 Flash Vision API → returns a structured conversion audit with a score, pass/fail analysis, and prioritized fixes. Shareable roast URLs act as a built-in viral loop.

**Stack:** `Next.js 15` `TypeScript` `Gemini 1.5 Flash` `Supabase` `Upstash Redis` `Vercel`

**Architecture highlights:**
- Serverless App Router backend with prompt chaining and structured JSON output parsing
- IP-based rate limiting via sliding window algorithm (Upstash Redis)
- Zero marginal cost: full infra on free tiers — $0/month to run
